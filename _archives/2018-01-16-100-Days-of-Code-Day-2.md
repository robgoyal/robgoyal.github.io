Day 2 of the 100 Days of Code and the challenge is getting more challenging. It's difficult to stay motivated to code every single day but I pushed myself to get it done and I succeeded. Somewhat.

<!--more-->

## Projects

#### WikipediaViewer

This is the 3rd project in the Intermediate Projects section of FreeCodeCamp. FCC is an amazing resource designed to teach web development. I'm not striving for a career in web development but I'm interested in learning Javascript as well as just learning to problem solve in an area that has become so prevalent in our use.

The task is to create a single page web application which will return search results similar to Wikipedia's search functionality, and provide little snippets about the resource. Another button will also exist which will link to a random Wikipedia article.

[Link](https://github.com/robgoyal/FreeCodeCamp/tree/master/FrontEndDevCert/IntermediateProjects/WikipediaViewer) to the project source code.

## Challenges Faced

I was only able to set up some boilerplate for the HTML and an empty Javascript file. Most of my time was spent playing around in the Wikipedia API sandbox since much of the information is quite dense to understand and maneuver around. I was able to get some information that resembled the results returned from the sample web page up on CodePen but I had trouble getting the links. Hopefully, I can return to it tomorrow and start setting up sample AJAX requests to perform the search results instead of using the sandbox.

## Lessons Learned

API documentation is hard to read and is definitely a lot of practice. It's especially difficult for a site like Wikipedia which consists of so much information.

The other thing I realized is how quickly information gets forgotten if you don't practice consistently. I was just scratching the basics of web development around 2 months ago but I've quickly forgotten a decent portion of it. I think going through a few resources and googling many of the questions I have will help me recollect what I knew before and hopefully learn new things as well.
